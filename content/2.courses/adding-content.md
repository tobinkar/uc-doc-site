# Adding Content to Courses

Learn how to add videos, documents, quizzes, and more to your courses.

## Content Types

| Type | Best For | File Types |
|------|----------|------------|
| Video | Lectures, demos, tutorials | MP4, MOV, WebM (max 5GB) |
| Document | Readings, handouts, slides | PDF, DOCX, PPTX |
| Text | Written lessons, instructions | Rich text editor |
| Quiz | Knowledge checks, exams | Built-in quiz builder |
| Assignment | Open-ended submissions | Learner uploads or text |
| Embed | External content | YouTube, Vimeo, others |

## Adding a Video Lesson

1. Inside a module, click **+ Add Lesson**
2. Select **Video**
3. Enter a lesson title
4. Upload your video:
   - Drag and drop, or click to browse
   - Supported: MP4, MOV, WebM
   - Max file size: 5GB
5. Wait for processing (usually 1-2 minutes)
6. (Optional) Add description, transcript, or downloadable resources
7. Click **Save Lesson**

### Video Best Practices
- Keep videos under 10 minutes for better engagement
- Use 1080p resolution for quality
- Include captions for accessibility
- Break long topics into multiple short videos

## Adding a Document Lesson

1. Click **+ Add Lesson** → **Document**
2. Enter a lesson title
3. Upload your file (PDF, DOCX, PPTX)
4. Choose display mode:
   - **Inline Viewer** — View within the platform
   - **Download Only** — Learners download to view
5. Click **Save Lesson**

## Adding a Text Lesson

1. Click **+ Add Lesson** → **Text**
2. Enter a lesson title
3. Use the rich text editor to write content:
   - Headers, bold, italic, lists
   - Insert images
   - Add links
   - Embed code blocks
4. Click **Save Lesson**

## Adding an Embed

Embed external content like YouTube videos or interactive tools.

1. Click **+ Add Lesson** → **Embed**
2. Enter the embed URL or paste embed code
3. Supported sources:
   - YouTube, Vimeo
   - Google Slides
   - Typeform, Calendly
   - Any iframe-compatible content
4. Click **Save Lesson**

## Organizing Content

### Reorder Lessons
Drag and drop lessons to change their order within a module.

### Reorder Modules
Drag and drop modules to change course structure.

### Move Lessons Between Modules
1. Click on the lesson → **⋮** → **Move to...**
2. Select the destination module
3. Click **Move**

## Content Visibility

Control when learners can access content.

### Draft Mode
New lessons are drafts by default. Learners can't see draft content.

1. Edit the lesson
2. Toggle **Published** to make it visible

### Drip Content
Release content on a schedule.

1. Go to **Course Settings → Drip Content**
2. Enable drip scheduling
3. Set rules:
   - **Days after enrollment** — Module 2 unlocks 7 days after signup
   - **On specific date** — Content available January 15th
   - **After completing previous** — Must finish Module 1 first
