# Creating Courses

A complete guide to building effective courses on Unicourse.

## Course Structure

```
Course
├── Module 1: Introduction
│   ├── Lesson 1: Welcome Video
│   ├── Lesson 2: Course Overview (Document)
│   └── Lesson 3: Pre-Assessment (Quiz)
├── Module 2: Core Concepts
│   ├── Lesson 1: Video Lecture
│   ├── Lesson 2: Supplementary Reading
│   └── Lesson 3: Knowledge Check (Quiz)
└── Module 3: Final Assessment
    ├── Lesson 1: Review
    └── Lesson 2: Final Exam (Quiz)
```

## Creating a New Course

1. Click **+ New Course** from the Dashboard
2. Fill in the basics:
   - **Title** — Clear, descriptive name
   - **Description** — What learners will gain (2-3 sentences)
   - **Category** — Help organize your course catalog
   - **Cover Image** — 1280x720px recommended
3. Click **Create Course**

## Course Settings

Access via the **Settings** tab within your course.

### General
- **Title & Description** — Update course details
- **Category** — Organize courses by topic
- **Visibility** — Draft (hidden) or Published (visible)

### Enrollment
- **Open Enrollment** — Anyone can join
- **Invite Only** — Require invitation to enroll
- **Enrollment Limit** — Cap the number of learners
- **Approval Required** — Review enrollments before granting access

### Completion
- **Required Lessons** — Which lessons must be completed
- **Passing Score** — Minimum quiz score to pass (e.g., 80%)
- **Certificate** — Issue certificate on completion

### Access
- **Time Limit** — Learners must complete within X days
- **Drip Content** — Release modules on a schedule
- **Prerequisites** — Require another course first

## Duplicating a Course

Create a copy to reuse structure and content.

1. From the course list, click **⋮** → **Duplicate**
2. Enter a new title
3. Choose what to copy:
   - ✅ Content (modules, lessons)
   - ✅ Settings
   - ❌ Learner data
4. Click **Duplicate**

## Archiving a Course

Remove a course from active use without deleting it.

1. Go to **Course → Settings**
2. Scroll to **Danger Zone**
3. Click **Archive Course**
4. Confirm

Archived courses:
- Hidden from learner catalog
- Existing learners retain access
- Can be restored anytime

## Deleting a Course

**Warning:** This permanently removes the course and all learner progress.

1. Archive the course first (required)
2. Go to **Settings → Archived Courses**
3. Find the course → **⋮** → **Delete Permanently**
4. Type the course name to confirm
5. Click **Delete**
