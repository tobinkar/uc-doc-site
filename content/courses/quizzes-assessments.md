# Quizzes & Assessments

Create quizzes to test knowledge and measure learner understanding.

## Quiz Types

| Type | Use Case | Grading |
|------|----------|---------|
| Knowledge Check | Reinforce learning mid-lesson | Optional, low stakes |
| Module Quiz | Test understanding after a section | Automatic |
| Final Exam | Comprehensive assessment | Automatic, gates certificate |
| Survey | Gather feedback (not graded) | None |

## Creating a Quiz

1. Inside a module, click **+ Add Lesson** → **Quiz**
2. Enter a quiz title
3. Configure quiz settings (see below)
4. Add questions
5. Click **Save Quiz**

## Quiz Settings

### Basic Settings
- **Title** — Name of the quiz
- **Description** — Instructions for learners
- **Time Limit** — Optional countdown timer (e.g., 30 minutes)

### Attempts & Scoring
- **Passing Score** — Minimum percentage to pass (e.g., 80%)
- **Attempts Allowed** — Number of retakes permitted
- **Show Correct Answers** — Reveal answers after submission
- **Randomize Questions** — Different order for each attempt

### Progression
- **Required to Continue** — Must pass before unlocking next lesson
- **Required for Certificate** — Must pass to receive certificate

## Question Types

### Multiple Choice
Learner selects one correct answer from options.

1. Click **+ Add Question** → **Multiple Choice**
2. Enter the question
3. Add answer options (minimum 2, maximum 6)
4. Mark the correct answer
5. (Optional) Add explanation shown after answering

### Multiple Select
Learner selects all correct answers.

1. Click **+ Add Question** → **Multiple Select**
2. Enter the question
3. Add options and mark all correct answers
4. Set partial credit (optional):
   - Full credit only if all correct
   - Or partial credit for partially correct

### True/False
Simple binary choice.

1. Click **+ Add Question** → **True/False**
2. Enter the statement
3. Mark whether it's true or false

### Short Answer
Learner types a text response.

1. Click **+ Add Question** → **Short Answer**
2. Enter the question
3. Add accepted answers (comma-separated)
4. Set matching mode:
   - Exact match
   - Contains (keyword matching)
   - Manual review required

### Essay
Open-ended response requiring manual grading.

1. Click **+ Add Question** → **Essay**
2. Enter the prompt
3. Set word limit (optional)
4. This question requires manual grading

## Question Settings

### Points
Assign point values to weight question importance.
- Default: 1 point per question
- Increase for harder questions

### Feedback
Add explanations shown after the learner answers.
- **Correct feedback** — Reinforces why the answer is right
- **Incorrect feedback** — Explains the correct answer

### Randomize Options
Shuffle answer choices to prevent memorization.

## Question Bank

Save questions for reuse across courses.

### Add to Question Bank
1. Create a question in any quiz
2. Click **⋮** → **Add to Question Bank**
3. Select or create a category

### Import from Question Bank
1. In quiz editor, click **Import from Bank**
2. Browse or search questions
3. Select questions to add
4. Click **Import**

## Viewing Results

### Individual Results
1. Go to **Course → Learners**
2. Click on a learner → **Quiz Results**
3. View their answers and score

### Quiz Analytics
1. Go to **Course → Analytics → Quizzes**
2. See overall pass rate, average score
3. Identify commonly missed questions
